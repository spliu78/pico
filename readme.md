# pico文件整理工具

根据日期，自动归档选定文件夹的文件，并跳过重复的文件

## cli工具

下载
`git clone https://github.com/spliu78/pico.git`

安装
`cd pico && npm install && npm build`

运行
`pico --help`

## 缘起

某天发现自己电脑里的各种资料太乱且到处是备份乱糟糟。。。闲下来的时候就先整理一下自己吧
